# Suggested Oracle APEX Pages

## Dashboard

- Employee petty cash balance.
- Available limit.
- Pending approvals.
- Open requests and recent expenses.

## Configurations

- Petty cash users and employee limits.
- Expense categories.
- Approval assignments.

## Petty Cash Request

- Create request.
- Submit request.
- Attach support documents.
- Approve or reject request.

## Petty Cash Expense

- Enter expense.
- Link expense to a request.
- Upload receipt.
- Submit for approval.
- Approve, reject, and account expense.

## Reports

- User balances.
- Expense summary.
- Request status.
- Audit log.
