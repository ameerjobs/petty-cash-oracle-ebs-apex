# Deployment Checklist

- Confirm custom schema name and object prefix.
- Confirm Oracle EBS operating unit and MOAC requirements.
- Confirm employee source from HRMS or local mapping.
- Confirm petty cash reimbursement accounting flow.
- Confirm AP invoice interface requirements.
- Configure APEX workspace, parsing schema, and authentication.
- Import APEX application export.
- Apply grants and synonyms required by the runtime schema.
- Compile all PL/SQL packages.
- Smoke test request, expense, approval, and reporting flows.
